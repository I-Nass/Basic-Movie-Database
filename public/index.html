Welcome to the Basic Movie Database!

This project allows you to **search for movies** using the OMDb API and **view their detailed information**. You can also keep track of your **recently viewed movies** in a history section.

This is a **work in progress** and is being continuously improved. You might notice some movies don't have posters or have slightly inconsistent data due to the external API source.

Enjoy exploring the world of movies!
